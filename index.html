MEAN Stack E-commerce App

I created this e-commerce app so that I could really digest how the MEAN stack worked together, from start to finish. As I started putting it together, I decided to style it using Angular Material and Flexbox, and I used the Snipcart API for user authentication and shopping cart functionality.

As a byproduct, I ended up learning much more about the app deployment process as I ran into many common issues while deploying to Heroku. Luckily, I was able to troubleshoot the issues and fix them.

Full-stack Web Application

This project allowed me to graduate from building static non-persistent websites to full-fledged web applications. For this specific application, I used Node.js for the server-side language, Express for route-handling, and MongoDB for the NoSQL database.

Weather App

This was my first time working with an API (openweathermap.org). Due to the newest geolocation restrictions on Chrome and the lack of free HTTPS support from the API, I couldn't automatically display the weather for Chrome users. However, I chose to still provide them with the best experience possible by adding a search bar that allowed the user to add the city name or zip code, which wasn't part of the original design.
